# RL TraderBot

📈 یک ربات معاملاتی با یادگیری تقویتی، متصل به متاتریدر 5 و رابط گرافیکی.

## نصب وابستگی‌ها:
pip install -r requirements.txt

## اجرای برنامه:
python main.py

## قابلیت‌ها:
- آموزش مدل با داده واقعی
- ذخیره/بارگذاری مدل
- ترید زنده با MT5
- رابط گرافیکی Qt
